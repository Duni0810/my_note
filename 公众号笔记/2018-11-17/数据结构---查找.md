# 查找

关于查找嘛，我觉得大部分人肯定都不陌生。像我是做嵌入式软件的，其实程序大部分都是对数据进行处理，处理中难免要进行数据查找，我们一般用的查找都是进行循环遍历的，例如：

``` c
	for(i = 0; i < len; i++ ) {
		if(arg == data[i]) {
			// idle
		}
	}
	
```
